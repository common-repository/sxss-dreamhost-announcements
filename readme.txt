== Description ==

You can send E-Mail newsletters with the [Dreamhost](http://sxss.nw.am/dreamhost/ "Dreamhost Webhosting - dreamhost.com") Announcement List feature from your Wordpress dashboard.

Languages: English, German. More informations on the official plugin page: [http://sxss.nw.am](http://sxss.nw.am "sxss.nw.am")

== Installation ==

Step 1 - Dreamhost:
Go to [the Dreamhost Panel](https://panel.dreamhost.com "Dreamhost Panel"), 'Mail > Announce Lists > Add New Announcement List', setup your announcement list. Then go to the [Web Panel API](https://panel.dreamhost.com/index.cgi?tree=home.api "Dreamhost Web Panel API") > check 'All announcement_list functions' and click on 'Generate a new API Key now!' at the bottom of the page. Now you get the required API Key.

Step 2 - Wordpress:
Upload the sxss Dreamhost Announcements plugin to your blog, activate it, go to 'Tools > sxss Dreamhost Announcements'. Enter the API Key (step 1) and save it. Now you can post announcements directly from your dashboard!

Thanks for using one of my plugins. Check out [my other plugins](http://profiles.wordpress.org/sxss/ "http://profiles.wordpress.org/sxss/") in the Wordpress repository!
